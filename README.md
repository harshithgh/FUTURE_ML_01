# FUTURE ML Task 01 - Sales Forecasting

## 📌 Problem Statement

The goal of this project is to predict future sales based on historical business data.

## ⚙️ Approach

* Cleaned and processed date-based sales data
* Created time-based features
* Used Linear Regression for forecasting

## 📊 Results

* Generated sales predictions for the next 30 days
* Evaluated model using Mean Absolute Error (MAE)

## 📈 Visualization

* Plotted actual vs predicted sales using Matplotlib

## 🧠 Skills Gained

* Time-series analysis
* Data preprocessing
* Machine learning modeling
* Business insights interpretation

## ▶️ How to Run

1. Install requirements:
   pip install pandas matplotlib scikit-learn
2. Run the notebook:
   sales_forecasting.ipynb
